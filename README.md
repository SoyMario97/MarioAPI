## Installation in Windows

```
$ npm i marioapi --s
```

## Installation in Glitch

```
$ pnpm i marioapi
```

### Available Variables
```js
mario.dog()
mario.fox()
mario.cat()
mario.panda()
mario.koala()
mario.meme()
mario.hug()
mario.smoke()
```

### Example
```js
const mario = require("marioapi")

//you can use another variable instead of mario.dog()
let img = await mario.dog()
//form async
console.log(img)
```
